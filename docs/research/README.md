# Research Documentation

Research documentation captures background reasoning, academic framing,
candidate algorithms, and future implementation options.

Research docs are intentionally exploratory. They are not binding protocol
contracts until a decision is promoted into `docs/reference/` or
`docs/dev/`.

## Documents

- [TEP Academic Research Plan](TEP_ACADEMIC_RESEARCH_PLAN.md)
- [TEP Research Map](TEP_RESEARCH_MAP.md)
- [TEP Reasoning Research](TEP_REASONING_RESEARCH.md)
- [TEP Planning And Curiosity Research](TEP_PLANNING_CURIOSITY_RESEARCH.md)

## Maintenance Rules

- Keep research docs in English.
- Cite sources when adding external research.
- Separate candidate ideas from accepted runtime behavior.
- Promote stable findings into reference or developer docs when they become
  implementation contracts.
